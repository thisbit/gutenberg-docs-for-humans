## Disable typography settings for core blocks (works with 5.9)
```
{
	"version": 2,
	"settings": {
		"typography": {
			"letterSpacing": false,
			"Appearance": false,
			"letterCase": false,
			"fontFamily": false,
			"fontSize": false,
			"fontStyle": false,
			"fontWeight": false,
			"lineHeight": false,
			"textDecoration": false,
			"textTransform": false,
			"dropCap": false,
			"customFontSize": false,
			"customLineHeight": false
		}
	}
}
```
